# Uhimilishaji Tanzania - Artificial Insemination Website

A professional, responsive website for Uhimilishaji Tanzania, providing artificial insemination services for cattle in Tanzania.

## Features

- **Responsive Design**: Fully compatible with all devices (mobile phones, tablets, laptops, desktops, and smart TVs)
- **Bilingual Support**: English and Kiswahili language options
- **Modern UI/UX**: Professional design with smooth animations and user-friendly interface
- **Comprehensive Services**: 
  - Artificial Insemination
  - Heat Detection
  - Pregnancy Diagnosis
  - Breed Consultation
  - Training for Farmers
- **Educational Resources**: Information about artificial insemination and its benefits
- **Contact Integration**: WhatsApp, phone, email, Facebook, and Instagram links

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)
- Google Fonts (Inter)

## Deployment

### GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Select the branch (usually `main` or `master`)
4. Select the folder (usually `/root`)
5. Click Save
6. Your site will be live at: `https://[username].github.io/[repository-name]`

### Custom Domain (Optional)

1. Add a `CNAME` file with your domain name
2. Configure DNS settings as per GitHub Pages documentation

## Local Development

1. Clone the repository:
   ```bash
   git clone [repository-url]
   cd [repository-name]
   ```

2. Open `index.html` in a web browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

3. Navigate to `http://localhost:8000` in your browser
